Sobre nosotros

Grupo formado por Cristian Gallardo, Anais Diaz y Matias Salas

Aqui presentamos nuestro proyecto final para Paradigmas de Programación.
Consta de una una pagina web creada para facilitar las compras tecnologicas que quiera realizar un usuario, extrae mediante un link la base de datos que almacena todos los productos que ofecen las grandes tiendas y se recopilan en un unico archivo "csv" donde se encontraran especificaciones como: Nombre del producto, la tienda donde se ofrece, el precio del producto, entre otros. De esta manera se facilita la comparacion de precios y el usuario no necesitará investigar de tienda en tienda lo que se quiera comprar.
